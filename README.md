# Bookmark
このアプリは、よく使用するリンクを保存するアプリです。

必要なリンクがすぐに見つけられなかったり、お気に入り等で保存していても乱雑になりがちなリンクを

簡単に保存、追加、削除できます。

- **Bookmarkアプリの３つの特徴**

  ①カテゴリー別に保存できる

  ②メモ機能がついている

  ③アプリ内で直接webページを開いて閲覧できるようにWebView機能がついている
